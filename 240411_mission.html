03 다음 코드에서 선택자가 어떤 태그를 선택하는지 설명하시오.

ex) div > a : div 태그 아래에 있는 a 태그를 선택한다.
  1. header a : header태그의 후손인자인 모든 a태그를 선택한다.
  2. eader > * > h1 : header 바로 직계 자손의 자손인 h1을 선택
  3. header > div:first-child > a : header 바로 직계 자손의 첫번째div 태그 직계 자손인 a 태그를 선택한다.
  4. header a:first-child > a : header태그의 후손 여러 a태그들 중 첫번째 a태그의 직계 자손인 a 태그를 선택한다.
  5. div > h1, h2 : div의 직계자손인 h1태그와, 일반 웹문서 전체를 통틀어 h2태그를 선택한다.
  6. .title.card :class별명이 title임과 동시에 class이름이 card이기도 한 태그 선택한다.

05 다음 코드에서 선택자가 어떤 태그를 선택하는지 적으시오.

<body>
  <header>
    <hgroup>
      <h1><a href="#">Header</a></h1>
    </hgroup>
    <nav>
      <div><a href="#"<Item</a></div>
      <div><a href="#"<Item</a></div>
      <div><a href="#"<Item</a></div>
    </nav>
  </header>
</body>

ex) 1번 선택자 구문이 의미하는 바 (nav a) : nav 태그 아래 모든 a태그를 선택한다.

  • 1번 선택자 (header a) : header안에 후손인 모든 a태그를 선택한다.
  • 2번 선택자 (header > * > h1) : header에 있는 직계 자손 안에 한번 더 타고 내려간 직계 자손 h1을 선택한다.
  • 3번 선택자 (header > * > *:first-child > a) : header안에 있는 /*html5ver.*/전체 요소중, a 태그가 선택된다.
  • 4번 선택자 (header a:first-child) :header 안에 있는 모든 첫번째 a태그를 선택한다.
